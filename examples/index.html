    Getting Started

    Installing

    You can install the grid via npm 

    
            npm install giga-grid --save

    You can also install the grid via the browser script tag 

    
            <script src="https://fb.me/react-0.14.6.js"></script>
            <script src="https://fb.me/react-dom-0.14.6.js"></script>
            <script src="giga-grid.js"></script>
            <!-- we will try to make font-awesome optional -->
            <link rel="stylesheet"
            href="https://maxcdn.bootstrapcdn.com/font-awesome/4.5.0/css/font-awesome.min.css">

    CommonJS

    The npm install creates giga-grid under node_modules 

    
            var GigaGrid = require("giga-grid").GigaGrid;

            var data = [{"col1": "Foo", ... }, ... ];
            var columnDefs = [{colTag: "col1", title:"Column 1"}, ... ];

            var myGrid = React.createElement(GigaGrid, { data: data, columnDefs: columnDefs });
            ReactDOM.render(myGrid, document.body);

    ES6 with JSX

    
            import {GigaGrid} from 'giga-grid';

            let data = [{"col1": "Foo", ... }, ... ];
            let columnDefs = [{colTag: "col1", title:"Column 1"}, ... ];

            ReactDOM.render(<GigaGrid data={data} columnDefs={columnDefs} />, document.body);

    Browsers

    We do not recommend this approach, as it pollutes the global namespace and require manual dependency management. Please consider a module loader!
